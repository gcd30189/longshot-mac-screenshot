## Mac 滚动截图的真实痛点

说正事之前，先聊聊截图这件事到底有多烦。

Mac 自带的 Shift + Command + 5，能截区域、全屏和窗口。普通用途够用，但遇到两种情况就歇菜了：一是要截一段很长的聊天记录或代码，二是要截取网页全文。这时候要么分多张截图再拼接，要么找浏览器插件凑合，反正都不顺手。

Mac 上的 **滚动截图**（也叫长截图）方案大概分这几类：

- **iShot**：支持自动滚动，操作顺手，但只支持竖向，横向截图暂无
- **Xnip**：滚动截图速度不能太快，免费版左上角有水印
- **Shottr**：自动滚动网页，但对第三方 App 的支持有限
- **Longshot**：手动控制滚动方向，支持竖向、横向和全景 360° 拼接

几者各有侧重。如果你的需求刚好落在「需要横向截图」「需要截取复杂背景的滚动区域」这两点上，Longshot 是目前很少几个能做到的选择。

## Longshot 核心功能实测

### 滚动截图：三个方向都能截

这是 Longshot 被提得最多的功能。

竖向滚动截图是标配，但 Longshot 还支持**横向滚动**和**全景 360° 拼接**——这两个在其他工具里很少见。横向滚动适合截取宽幅表格或横版设计稿，全景模式则能把整个桌面或超宽画面一网打尽。

另一个细节：截图时可以**动态调整滚动方向**，滚到一半发现方向不对，随时改，不用重来。非纯色背景、包含固定元素（比如顶部导航栏）的区域，它的拼接算法也能处理得比较干净。

从知乎截图横评来看，Longshot 独有丰富的测距功能，对于设计等岗位来说确实很赞，而且回溯功能也很棒——截图 N 个物品时，通过回溯能保证每次截图的大小尺寸、位置都一致，截出来很统一。

### 标注工具：一套够用的基础配置

截完图直接可以标注，不用跳到另一个软件。工具包括：矩形、椭圆、线条、箭头、文字、马可笔，还有步骤标注（带序号的气泡，做教程截图很有用）和马赛克。

像素取色和屏幕测距放在同一个界面里，对设计师来说比较省事——截图、取色、量距离，一套下来不用切软件。

### 贴图钉图：把截图变成桌面便签

这个功能有点意思。截完图之后，可以把它「钉」在桌面最上层，缩放、旋转随意，双击关闭。用来对比两张图、参考一段代码、或者临时把某个数据贴在旁边，不用来回切窗口。

剪贴板里的图片、文本、HTML 都可以直接贴图，不限于截图。

### OCR 文字识别：离线运行，隐私友好

框选屏幕区域，识别文字，复制到剪贴板。Longshot 使用离线 OCR 算法，注重隐私保护，能迅速提取图片、视频、文档等任何地方的文字，还支持 QR 码和条形码识别，以及语音播报识别到的文字。

离线意味着不用联网就能用，识别结果不会上传服务器，对处理敏感内容的人来说是个加分项。

免费版每天可以 OCR 12 次，专业版无限制。
